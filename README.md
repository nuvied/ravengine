# Ravengine

A simple, lightweight game engine by Blue Raven.

## About

Ravengine is a straightforward game engine built in C++ designed to help developers create 2D and 3D games with ease. It provides essential game development tools and utilities while maintaining a clean, accessible codebase.

## Features

- **Lightweight Design** - Minimal dependencies and streamlined architecture
- **C++ Foundation** - Modern C++ implementation for performance and reliability
- **Cross-Platform** - Designed to run on multiple operating systems
- **Easy to Use** - Intuitive API for quick game development
- **Extensible** - Modular design for easy customization and feature additions

## Getting Started

### Prerequisites

- C++ compiler (C++11 or later)
- CMake (for building)
- Git

### Installation

Clone the repository:
```bash
git clone https://github.com/nuvied/ravengine.git
cd ravengine
```

Build the engine:
```bash
mkdir build
cd build
cmake ..
make
```
